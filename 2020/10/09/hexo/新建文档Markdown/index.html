新建文章的方法

(1)通过命令新建

$ hexo new 'title' # 新建文章
$ hexo new draft 'draftname' #新建草稿

（2）编辑Markdown文档
执行上述命令后，会在仓库文件夹下的/source/_posts中找到新增的Markdown文档。

（3）生成，push

$ hexo g
$ hexo s
$ git add -A
$ git commit -m '说明'
$ git push origin master

Markdowm添加图片 - 图床功能

  • imgur：老牌稳定图床，国外服务器，慢
  • 腾讯云、阿里云
